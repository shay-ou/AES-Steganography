What is AES?
AES (Advanced Encryption Standard) is a symmetric key encryption algorithm. It encrypts your message using a 16-byte key so that it becomes unreadable ciphertext.
What is LSB Steganography?
LSB (Least Significant Bit) replaces the last bit of each pixel's R, G, B channel with one bit of the hidden message. This change is invisible to the human eye.
Why combine both?
Encryption alone reveals a message exists. Steganography alone isn't secure. Combined, the data is both unreadable AND undetectable — dual-layer protection.
